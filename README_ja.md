# WebSurf Helper

Web サーフィンを快適にする小さな Chrome 拡張です。機能は 2 つだけ。

1. **リンクをダブルクリック**で新しいタブで開く（フォアグラウンド / バックグラウンドは設定可能）
2. **上下キーのスクロール量**を自分好みに（デフォルトはビューポート半分。1 行スクロールと PageUp/PageDown の中間）

## 前提条件

- **Manifest V3** に対応した Google Chrome（または Chromium 系ブラウザ）

## インストール（パッケージ化されていない拡張機能として読み込み）

1. `chrome://extensions/` を開く
2. 右上の **デベロッパーモード** を ON
3. **「パッケージ化されていない拡張機能を読み込む」** をクリックし、このディレクトリ (`websurf-helper/`) を選択
4. すべてのページで機能が有効になる

## 設定

`chrome://extensions/` → 拡張機能の **「詳細」** → **「拡張機能のオプション」** から設定画面を開く。

| 設定項目 | デフォルト | 説明 |
|---------|---------|------|
| リンクのダブルクリックを有効にする | ON | ダブルクリックで新タブを開く機能の ON/OFF |
| 新しいタブをすぐに表示する | OFF | OFF だとバックグラウンドで開く |
| 上下キーのカスタムスクロールを有効にする | ON | スクロール幅変更機能の ON/OFF |
| スクロール量（ビューポート比） | 0.5 | キー 1 押下あたりのスクロール量（0.1 〜 1.0） |

## 動作の補足

- `<input>` / `<textarea>` / `<select>` / `contenteditable` にフォーカスがある時はカスタムスクロールが無効になる
- 修飾キー（Ctrl / Shift / Alt / Cmd）併用時はカスタムスクロールに介入しないので、サイト固有のショートカットも壊れない
- `javascript:` から始まるリンクや、ページ内アンカー（`#...`）はダブルクリック処理の対象外
